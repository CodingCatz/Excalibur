# 隱私權政策 / Privacy Policy

最後更新：2025 年 6 月

## 資料收集

Excalibur — Game Design Studio（以下簡稱「本工具」）**不收集任何個人資料**。

- 不使用分析工具（Analytics）
- 不使用廣告追蹤
- 不設追蹤 Cookie

## 本地儲存

您的專案資料儲存於瀏覽器 IndexedDB（本機），未經您主動操作不會離開您的裝置。

## Google Drive 連線

當您選擇登入 Google 帳號進行雲端備份時：

- 本工具僅申請 `drive.file` 權限，即**只能存取由本工具建立的檔案**
- 無法讀取您 Google Drive 中的其他檔案
- 備份內容以 ZIP 格式儲存在您 Drive 的 `Excalibur/` 資料夾中
- 您可隨時在 Google 帳號設定中撤銷授權

## 第三方服務

本工具為純前端靜態網頁應用程式（PWA），託管於 GitHub Pages，不存在後端伺服器。除 Google Drive API 外，不與任何第三方服務共享資料。

## 聯絡方式

如有疑問，請至 GitHub 提交 Issue。
